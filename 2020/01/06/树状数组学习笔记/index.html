树状数组学习笔记

Posted on 2020-01-06 Edited on 2020-01-17 In 数据结构 , 算法 , pat , leetcode

fenwick数组（树状数组）下标从1开始数起～

lowbit(x)

lowbit(x) = x & (-x) 含义是能整除x的最大2的幂次（原理是补码：取反加一）

fenwick tree

作用

定义数组c[i]

c[i]的覆盖范围是lowbit(i)

c[x] = a[x - lowbit(x) + 1] + …+ a[x]

sum(1,x) = a[1] + … + a[x] = a[1] + … + a[x-lowbit(i)] + c[x] = sum(1, x - lowbit) + c[x]

时间复杂度logN

update(x, v)

把所有覆盖a[x]的c[i]加上v，数学上就是x不断加上lowbit(x), 证明算法笔记p474

时间复杂度logN

应用

求出序列中比它小的数的个数

给定一个有N个正整数的序列A，对于序列中的每一个数，求出序列中比它小的数的个数

解法定义一个hash[a[i]], 扫描到a[i], 将hash[a[i]]++；

统计比a[i]小的个数只需要getSum[a[i]-1];

实际写的时候hash[]不必要真正出现

统计左边比自己大：getSum(N)-getSum(A[i])

统计右边比自己小\大：从右边往左边扫描

补充：假如a[i] 过于大，以至于c[]数组开不了那么大，可以采用离散化处理（即排序～）

第k大

用hash[i] 来记录每个元素出现的次数

求k最大就是在求最小的i使得 hash[1]+…+hash[i] >= k

等价于第一个 getSum(i) >= k的i; 在搜寻i的时候还可以使用二分搜索

二维矩阵、二维树状数组

相求A[a][b]和A[x][y]之间的 所有元素之和可以计算getSum(x, y) - getSum(a-1, 1) - getSum(1, b-1) + getSum (a-1,b-1)
